    vuePress-theme-reco yzhe819    2020 - 2021

    快速构建简洁又优雅的个人博客

    yzhe819 7/6/2021 搭建

    👉如何用VuePress快速搭建一个和博主一样的博客网站？请看这里！！

    # 前言

    作为记性较差的学生，我一直想要一个文档类的技术博客，所以为了更加适合用于自己查阅技术文档和记录各种零散的知识。在攒了一堆笔记后，觉得不能再懒下去了，经过一番搜索，最终选择使用vuepress加上vuepress-theme-reco主题进行搭建。首先说明搭建这个博客是非常简单及快速的，预计耗时为2个小时😵。(主要是找材料的时间，不够根据下列步骤应该能更快的完成）

    # 介绍

    VuePress (opens new window)是一个Vue 驱动的静态网站生成器，由尤大开发的，支持vue语法。

    vuepress-theme-reco (opens new window)则是vuepress的主题，个人觉得是最好看的一个（满足了简洁和功能的要求）。说了这么多，直接开始上手。

    # 安装

    # 前置环境安装

    • 需要node环境和npm支持，这里不做讲解

    • 如果不会安装npm请转到：安装指引 (opens new window)

    # 安装初始化

    • 安装
    npx @vuepress-reco/theme-cli init my-blog
    1
    • 然后进入views文件夹内，新建一个markdown文件，比如Hello.md
    ---
title: Hello world
date: 2020-07-06
sidebar: 'auto'
categories:
 - 博文
tags:
 - 日常
---
::: tip
你好
:::

    • 跑起来
    npm run dev
    1
    • 然后我们就能看见我们的博客已经跑起来了，尽管内容还有很多要改，但第一步已经成功了🎉
      • 这里介绍一下在markdown文件内的被---包围的部分是 YAML front matter (opens new window) (opens new window)，这部分是用来设置标题头，种类，标签，侧边栏等等内容的部分，比如sidebar: 'auto'代表开启自动侧边栏。
      • 按照规定以上的内容会被显示成摘要
      • 剩下的就是我们markdown的正文了
