# yzhe819.github.io

### 介绍：

一个平平无奇的个人博客，用来存放一下编程之路的笔记文件



笔记主要是四大课程的内容：

- 🔗    计算机网络
- ⚙    操作系统
- 💊    数据结构与算法
- 🖥    计算机组成



### 运行：

> master分支：打包后的文件
> blogcode分支：blog原文件



#### 项目设置

```
git clone https://github.com/yzhe819/yzhe819.github.io.git
npm install
```

#### 运行

```
npm run serve
```

#### 构建

```
npm run build
```
