# yzhe819.github.io

### 介绍：

一个平平无奇的个人博客，用来存放一下编程之路的笔记文件

使用了[VuePress](https://vuepress.vuejs.org/)和[vuepress-theme-reco](https://vuepress-theme-reco.recoluan.com/)搭建



笔记主要是四大课程的内容：

- 🔗    计算机网络 ✔
- ⚙    操作系统 
- 💊    数据结构与算法
- 🖥    计算机组成



### 运行：

> 1. master分支：打包后的文件
> 2. blogcode分支：blog原文件



#### 项目设置

```shell
git clone https://github.com/yzhe819/yzhe819.github.io.git
git checkout blogcode
npm install
```

#### 运行

```shell
npm run serve
```

#### 构建

```shell
npm run build
```
